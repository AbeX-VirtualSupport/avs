# AbeX Virtual Solutions Portfolio

A responsive, single-page portfolio website for Abegail Mercado.

## Files
- `index.html` — website content and structure
- `styles.css` — responsive design and branding
- `script.js` — mobile menu, filters, animation, and image previews
- `assets/` — project screenshots, profile image, and certificates

## How to view
Open `index.html` in a web browser.

## Deploy to GitHub Pages
1. Create a new GitHub repository.
2. Upload all files and the `assets` folder.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, select **Deploy from a branch**.
5. Choose the `main` branch and `/root`, then save.
